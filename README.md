## 功能特点


### 🔍 搜索功能
- 支持关键词搜索Hanime1视频
- 支持视频链接直接解析
- 显示视频封面、标题、ID等详细信息

### 📥 下载功能
- 支持多线程下载，提高下载速度
- 支持同时下载多个视频，可配置最大同时下载数
- 支持断点续传
- 支持选择下载画质
- 支持自定义文件名命名规则
- 支持跳过已存在文件

### 📋 下载管理
- 可视化下载队列
- 支持暂停/恢复/取消下载
- 实时显示下载进度和速度
- 每个任务独立显示进度百分比
- 下载历史记录

### ⚙️ 配置选项
- 手动设置Cloudflare Cookie
- 配置下载路径
- 选择单线程/多线程下载
- 设置线程数量
- 设置最大同时下载数
- 配置文件名命名规则
- 选择是否覆盖已存在文件

### 🎨 界面设计
- 简洁直观的用户界面
- 响应式布局
- 清晰的进度显示
- 状态栏实时提示

## 安装说明

### 方法一：直接运行EXE文件
1. 从Release页面下载最新的`Hanime1Download.exe`文件
2. 双击运行即可，无需安装Python环境

### 方法二：从源代码运行
1. 克隆或下载本项目
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
3. 运行主程序：
   ```
   python hanime1_gui.py
   ```

## 使用方法

### 1. 搜索视频
- 在搜索框中输入关键词，点击搜索按钮
- 或直接粘贴Hanime1视频链接，程序会自动解析

### 2. 查看视频详情
- 点击搜索结果中的视频，查看视频详情
- 可以看到视频的不同画质源

### 3. 添加到下载队列
- 在视频详情页面，点击"下载"按钮，选择画质
- 视频会被添加到下载队列

### 4. 开始下载
- 点击"开始下载"按钮，程序会根据设置的最大同时下载数开始下载
- 可以在下载列表中查看每个任务的进度

### 5. 管理下载
- **暂停下载**：点击"暂停下载"按钮，暂停所有当前下载
- **恢复下载**：点击"恢复下载"按钮，恢复所有暂停的下载
- **取消下载**：点击"取消下载"按钮，取消所有当前下载并删除临时文件
- **清空列表**：点击"清空列表"按钮，清空下载队列

### 6. 配置设置
- 点击"设置"按钮，打开设置对话框
- 根据需要配置各项参数
- 点击"保存"按钮保存设置

## Cloudflare Cookie设置

由于Hanime1使用Cloudflare保护，你可能需要手动设置Cloudflare Cookie：

1. 在浏览器中访问Hanime1网站
2. 登录并完成Cloudflare验证
3. 复制浏览器中的`cf_clearance` Cookie值
4. 在设置对话框中粘贴到"Cloudflare Cookie"输入框
5. 点击"保存"按钮

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 下载方式 | 选择单线程或多线程下载 | 多线程 |
| 线程数量 | 多线程下载时的线程数 | 4 |
| 最大同时下载数 | 同时进行的下载任务数 | 2 |
| 下载画质 | 批量下载时的默认画质 | 最高 |
| 下载路径 | 视频保存路径 | 当前目录/hanimeDownload |
| 文件名命名规则 | 支持{title}和{video_id}变量 | {title} |
| 覆盖已存在文件 | 是否覆盖同名文件 | 否 |

## 常见问题

### Q: 搜索时提示"有Cloudflare检测"？
A: 请手动设置Cloudflare Cookie，参考"Cloudflare Cookie设置"部分。

### Q: 下载速度慢？
A: 可以尝试增加线程数或最大同时下载数，但请注意不要设置过高，以免被服务器限制。

### Q: 下载失败？
A: 检查网络连接，确保Cloudflare Cookie有效，或尝试更换下载画质。

### Q: 程序崩溃？
A: 请确保使用最新版本，或尝试从源代码运行查看错误信息。
